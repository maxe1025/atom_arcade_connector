# Information
This extension requires Godot version 4.5

# Setup for development
- Clone the repository and open it with your prefered IDE.
- You should use gpp as the compiler and SCons as the buildtool.
- Download the godot cpp files from their official github.
- Extract the folder inside this projects root folder and name it "godot-cpp".
- To build the extension enter "scons" in the command line.
